# Algo-Strategy-Lab

Repository for https://replit.com/@rejakayap/Algo-Bot

This is a TypeScript-based algorithmic trading strategy lab featuring backtesting capabilities for BTC/USDT futures trading strategies.

## Quick Start

- Visit the live Replit environment: https://replit.com/@rejakayap/Algo-Bot
- Run `pnpm install` to install dependencies
- Follow the commands in `replit.md` for development and deployment

## Technology Stack

- **Monorepo**: pnpm workspaces
- **Language**: TypeScript 5.9
- **Backend**: Express 5 with PostgreSQL + Drizzle ORM
- **Frontend**: React with Vite
- **Trading**: 51+ algorithmic strategies across 6 categories
- **Deployment**: Replit, Railway, Docker

For detailed information, see [replit.md](./replit.md)
